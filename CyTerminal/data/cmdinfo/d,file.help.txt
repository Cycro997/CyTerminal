d,file (subcategory)
Info
    prefix: ,file
    This command subcategory is for file handling related commands
Command list
    d,file.read <file> - reads file - [.r]
    d,file.write <file> <contents> - overwrites file - [.w]
    d,file.delete <file> - deletes file - [.d]
    d,file.readtovar <variable> <file> - makes the value of variable become file - [.rtv]
